🚨 문제 정의

숏폼 중심의 콘텐츠가 만드는 새로운 문해력 위기

증가하는 집중력 감소

30초 숏폼에 익숙해진 뇌는 30분 분량의 콘텐츠를 감당하지 못합니다

문해력 하락

"심심한 사과", "금일" 같은 기본 단어도 인식하지 못하는 현상 증가

학습 의욕 저하

5줄 이상의 글 읽기를 회피하는 습관이 강화됨

숏폼이 일상화되면서 호흡이 긴 콘텐츠를 이해하는 능력이 급속도로 감소하고 있습니다. 이는 단순한 개인의 문제가 아닌, 정보 활용 능력과 비판적 사고 기능에까지 영향을 미치는 사회적 위기입니다.

  • 성장 욕구: 하지만 자신의 독해력을 키우고 싶은 의지가 있음

📚 단계별 학습 과정

점진적 난이도 상향으로 독해력과 어휘력을 동시에 키우는 방식

1

핵심 추출 단계

긴 글에서 가장 중요한 것만 빠르게 파악

사용자가 처음 글을 받으면 AI가 100자 이내의 한 줄 요약으로 정보를 압축해 제공합니다. 긴 글을 읽기 전에 "이 글이 뭘 말하는 건가"를 빠르게 파악하도록 도와줍니다.

제공 정보:
  • 100자 이내 한 줄 요약
  • 핵심 키워드 3~5개
  • 예상 읽기 시간
  • 글의 난이도 평가
2

용어 학습 단계

모르는 단어를 즉시 해결하며 어휘력 확장

글 속의 낯선 용어와 전문 개념을 자동으로 감지합니다. 사용자 눈높이에 맞춘 설명을 제공하고, 해당 용어들을 개인 용어집에 저장하여 반복 학습 시 재사용합니다.

학습 기능:
  • 문맥에 맞는 단어 설명
  • 예시 문장과 함께 제시
  • 개인 용어집 자동 저장
  • 관련 단어 추천
3

원문 비교 단계

원문과 요약을 나란히 보며 관계 파악

좌우 분할 화면으로 원문과 요약을 동시에 표시합니다. 요약이 원문의 어느 부분에서 나온 것인지 색상으로 매핑하여 "글의 어떤 부분이 중요한지" 직관적으로 이해할 수 있습니다.

시각화 기능:
  • 원문 ↔ 요약 좌우 분할
  • 색상 매핑으로 대응 부분 표시
  • 섹션별 요약 비교
  • 주요 문장 하이라이트
4

이해도 확인 단계

비판적 사고 질문으로 수동적 읽기 극복

요약을 읽은 후 1~3개의 질문을 제시합니다. "글쓴이의 의도는?", "이 주장의 근거는?", "빠진 부분은?" 같은 질문을 통해 단순 정보 습득에서 능동적 사고로 전환합니다.

질문 유형:
  • 글쓴이의 의도 파악
  • 주장과 근거 검증
  • 빠진 논리 발견
  • 다른 관점 제시
5

직접 요약 단계

AI 없이 스스로 글을 읽고 정리

타이머와 빈 화면만 제공하고 사용자가 직접 요약을 작성하도록 합니다. 완성 전까지 어떤 힌트도 주지 않아 진정한 독해와 이해를 강제합니다. 이 단계부터 능동적 학습이 시작됩니다.

학습 방식:
  • 제한 시간 내 요약 작성
  • 완전한 독서 경험
  • 자기 언어로 정리
  • 집중력 강화 훈련
6

첨삭 피드백 단계

사용자 요약을 상세 분석하며 성장 방향 제시

사용자가 작성한 요약을 원문과 비교 분석합니다. 빠진 쟁점, 오해한 부분, 잘 쓴 문장을 구체적으로 표시합니다. 단순 점수가 아닌 성장 방향을 피드백으로 제공하여 동기 부여와 학습 효율을 동시에 달성합니다.

피드백 내용:
  • 빠진 핵심 요소 지적
  • 오해한 부분 설명
  • 잘 쓴 표현 칭찬
  • 다음 단계 학습 방향
7

레벨업 및 성장 단계

축적된 데이터로 자동 난이도 조정

읽기 패턴, 퀴즈 성과, 요약 품질을 종합 분석하여 사용자 레벨을 판정합니다. 레벨이 올라가면 요약 길이가 점진적으로 증가하고(3문장 → 5문장 → 한 단락), 새로운 학습 기능이 잠금 해제됩니다. 이를 통해 지속적인 성취감을 경험합니다.

레벨업 기준:
  • 읽기 지속 시간 증가
  • 퀴즈 정답률 향상
  • 요약 품질 개선
  • 어휘 습득 수 누적
